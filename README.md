For quickly create NodeJs module for helpers using this command:

```bash
npx degit NikolayMakhonin/nodejs-template-helpers <app name> && cd <app name> && npm i && npm run test
```

Module usage:
```bash
  <package.json>
  ...
  "dependencies": {
    "helpers": "file:../common/modules/helpers"
  },
  ...
```
